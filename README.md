# 🔊 Speaker Research Log

スピーカー・音響業界の最新ニュース・技術トレンド・研究論文を毎日自動調査し、日本語Markdownレポートとして蓄積するリポジトリです。

## プロジェクト概要

このリポジトリは、**Claude（Anthropic）のスケジュールタスク機能**を利用して、スピーカー・音響業界に関する包括的なリサーチレポートを**毎日自動生成・push**しています。

各レポートは英語・日本語の複数ソースからWebリサーチを行い、業界ニュース・技術動向・学術論文を横断的にカバーします。すべての情報には出典リンクが付いており、一次情報へのアクセスが可能です。

## レポートの構成

各日次レポートは以下のセクションで構成されています。

| セクション | 内容 |
|-----------|------|
| **Frontmatter** | 日付・テーマ名・タグ（5〜10個）・200字以内の要約（YAML形式） |
| **今日のハイライト** | その日の最も重要なニュース・発見を箇条書きで速報 |
| **目次** | レポート内の各セクションへのリンク |
| **特集** | 毎回異なるテーマを深掘りする中核セクション（2000〜3000字程度） |
| **業界ニュース** | 新製品・企業動向・イベント情報など |
| **技術トレンド** | 注目される技術的進展・標準化動向 |
| **論文・特許** | 学術論文・特許出願の紹介と解説 |
| **未来予測**（任意） | 中長期的な技術・市場の展望 |

## 自動生成の仕組み

1. **Claude のスケジュールタスク**が毎日自動で起動します
2. 英語・日本語のWeb検索を8〜10回実施し、最新情報を収集します
3. 必要に応じて個別記事をWebFetchで深掘りします
4. 過去のテーマインデックス（`index/by-topic.md`）を参照し、未カバーのテーマを特集として選定します
5. Markdown形式のレポートを生成し、`reports/YYYY/MM/YYYY-MM-DD.md` に配置します
6. トピック別インデックスを更新します
7. GitHub リポジトリに自動で commit & push します
8. GitHub Actions がpushを検知し、Issue #1 に通知コメントを投稿します

## 通知

レポートのpush時に GitHub Actions（`.github/workflows/notify-on-push.yml`）が自動実行され、Issue #1 に新規レポートの通知が投稿されます。Issue #1 をWatch/Subscribeすることで、新しいレポートの公開をメールで受け取れます。
