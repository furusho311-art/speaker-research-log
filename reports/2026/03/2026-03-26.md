## 🎯 特集：スピーカー製造の三大変革軸 {#特集}

### 変革軸①　AI駆動の品質管理と工場自動化

スピーカー製造における最大の変革は「**AIビジョン検査**」の本格導入である。従来、ボイスコイル巻線の検査や振動板の目視確認、接着剤の塗布量チェックは人手に依存していた。2025〜2026年にかけて、コンボリューショナルニューラルネットワーク（CNN）ベースのAI画像認識が生産ラインに組み込まれ始め、μmオーダーの欠陥を高速かつ高精度に検出できるようになっている。

GoertekのiPhone 16eスピーカー生産ラインはその最先端例だ。2025年3月、AppleのCOO Jeff Williamsが青島・濰坊の工場を視察した際、以下の技術を直接確認したと報告されている：

- **AIビジョン検査システム**：外観欠陥の自動スクリーニング
- **溶接ヘッド寿命予測AI**：設備の予知保全（ダウンタイムを大幅短縮）
- **レーザーはんだボール溶接技術**：従来比でメンテナンス停止時間を有意に削減
- **自社開発ボイスコイル巻線機**：髪の毛ほど細いエナメル線を高速・高精度に巻く専用機

Williams COOは「Goertekのスマートで効率的な製造プロセス」を高く評価したと伝えられている（[Patently Apple](https://www.patentlyapple.com/2025/03/apples-coo-visited-chinas-goerteks-plant-and-learned-about-the-companys-breakthroughs-in-automation-and-ai-technologies.html)、[Goertek公式](https://www.goertek.com/en/content/details15_235209.html)）。

AirPods用高光沢白色部品では、金型技術とインジェクション成形を組み合わせて「ゼロ欠陥生産」を実現。同社の濰坊光電パーク第2期は約2万人が働く大規模インテリジェント製造拠点に成長している。

#### TV用スピーカー全自動生産ライン

汎用の製造装置レベルでも高度化が進む。PLC＋タッチスクリーンを中核制御とし、振動板フィーダー自動投入→サーボプレスによる磁気回路部品精密圧入→SCARAロボットによるボイスコイル組付け・搬送→機械ビジョンによる欠陥スクリーニング・組立アライメント補正という流れで、「部品投入から梱包まで無人」が現実的な目標になっている（[JiuJu Auto事例](https://www.jiujuauto.com/en/case-center/case-study-of-tv-speaker-automation-production-line/6027.html)）。

> **重要な留意点**：スピーカードライバー単体の組立は自動化が進んでいる一方、「完成スピーカーの箱への収納・組み立て」はキャビネット内部での作業複雑性からロボットが対応困難で、まだ主に人手に頼る状況が続いている。

### 変革軸②　3Dプリント（AM）によるスピーカーエンクロージャーの量産

2026年初頭、英国のNode Audioが**Atom 525/650**シリーズを量産ローンチし、3Dプリント製スピーカーの本格量産時代の幕を開けた。最大の特徴は二つの特許技術：

**HTL（Helical Transmission Line：螺旋型トランスミッションライン）**
- ウーファーの音響エネルギーを中央チャンバーに導き、精密設計された螺旋状音道を経由させることで、キャビネットサイズをはるかに超えるバス応答を実現。
- 螺旋形状は3Dプリントなしには製造不可能な複雑ジオメトリ。

**MonoCell制振構造**
- 内部ダンピングのウール・フェルト材を廃止し、細かなリジッド格子構造（ラティス）で代替。
- 格子構造がブレースとして機能し、超剛性モノコックエンクロージャーを構成、キャビネット共振由来の着色を大幅低減。

製造装置として選択されたのが**Bambu Lab**のFDM（溶融積層造形）プリンター群。Node Audioは同社のH2Dとフリートの P1Sプリンターをインハウスで運用し、ポリマーAMによる4つのコア構成部品を量産している（[DEVELOP3D記事](https://develop3d.com/3d-printing/under-the-covers-of-node-audios-atom-speaker)、[Additive Manufacturing Media](https://www.additivemanufacturing.media/articles/loudspeaker-enabled-by-polymer-3d-printing-pic-of-the-week)）。

従来のスピーカー製造では「形状の自由度よりコスト・量産性」が優先されていた。AMの進化により、以下が変化している：

| 比較軸 | 従来製造（MDF/射出成形） | AMによる製造 |
|-------|----------------------|------------|
| 形状の自由度 | 低（金型制約） | 極めて高い |
| 内部音道設計 | 単純形状のみ | 螺旋・ラビリンス・フラクタルも可能 |
| 試作→量産コスト | 金型費用が大きい | 金型不要、小ロット対応 |
| 素材 | MDF, ポリマー, ダイキャスト | SLS用PA12, FDM用PLA/PETG/CF-PETGなど |
| 実績事例 | 大量生産向き | Node Audio Atom, Hylixa, 研究機関試作 |

### 変革軸③　サステナブル製造とグリーンマテリアル

大手ブランドも環境対応を製造設計に織り込み始めた：

- **JBL Go 3 Eco / Clip 4 Eco**：消費後リサイクルプラスチック最大90%、グリルに100%リサイクルファブリック使用（[Play it Green](https://playitgreen.com/sustainable-speakers-redefining-audio-excellence-with-sustainability/)）
- **House of Marley**：竹・責任調達木材・リサイクルアルミ・REWIND™ファブリック（リサイクルPET＋オーガニックコットン＋麻の混紡）採用
- ライフサイクルアセスメント（LCA）を活用して製造フェーズの炭素排出を可視化・削減する動きがメーカー・ODM双方で広がっている

## 📰 業界ニュース：サプライチェーン再編 {#ニュース}

### Goertek、ベトナムに最大3.91億ドル投資

GoertekはAppleサプライチェーン多様化の要請に応じ、ベトナムに最大3.91億ドルを投資して完全子会社を設立、AirPods・スマートウォッチ・XRデバイス向けの製造拠点を構築している。さらに**Foxconn Interconnect Technology**がGoertek Electronics Vietnamのベトナム工場を共同出資形式で買収。New Wing Interconnect Technology（北江省）が5,000万ドルを出資した（[South China Morning Post](https://www.scmp.com/tech/tech-trends/article/3248542/airpods-maker-goertek-open-vietnam-factory-part-apples-supply-chain-diversification)、[Apple Insider](https://appleinsider.com/articles/24/01/16/goertek-wants-to-build-a-new-factory-in-vietnam-for-airpods-apple-vision-pro)）。

### Foster Electric、車載スピーカーを中国からベトナムへ

フォスター電機（東証プライム上場）は、地政学リスク（米中経済摩擦）への対応として、**EV・自動車向けスピーカー生産を中国工場からベトナムへ移管**する方針を進めている。同社はすでにベトナムに複数の現地法人（Foster Electric Vietnam・Da Nang・Quang Ngai・Bac Ninh）を保有しており、段階的な移管を進める体制が整っている（[日刊工業新聞](https://www.nikkan.co.jp/articles/view/00753640)、[フォスター電機公式](https://www.foster.co.jp/about/group/index.html)）。

### GoertekのCES 2026展示：LBSプラットフォームと折りたたみ端末向けFプラットフォーム

2026年1月のCES 2026にて、Goertekは以下の音響技術を発表（[PR Newswire](https://www.prnewswire.com/news-releases/goertek-showcases-full-stack-innovations-in-acoustics-and-sensing-at-ces-2026-302656965.html)）：

- **LBS（Lateral Bass Speaker）プラットフォーム**：オープンイヤー型スピーカーの音量不足を解決する、デュアルラジエーター・片面放音構造。プライバシー保護のための遠距離フィールドキャンセレーション機能も統合し、音漏れを有効抑制。
- **超薄型大音量Fプラットフォーム**：折りたたみスマートフォン・タブレット向けの業界初となる超薄型高音量スピーカーモジュール。
- **Goermicro製 MEMS音響センサー**：第3世代防水アーキテクチャを採用した超小型（2.75×1.85mm）IP68対応MEMS音響センサー。スマートウォッチ・スマートフォン向け。

## ⚙️ 技術トレンド：AI品質管理・自動化 {#技術}

### AIビジョン検査の実装技術

2025年時点でのAI品質検査システムの技術的基盤（[Intelgic](https://intelgic.com/quality-inspection-trends-in-manufacturing-using-ai-in-2025)、[Overview.ai](https://www.overview.ai/blog/ai-inspection-systems-manufacturing/)）：

| 検査項目 | 活用技術 |
|---------|---------|
| ボイスコイル巻線ムラ | CNNによる高解像度カメラ画像の異常検知 |
| 接着剤塗布量 | カラーセグメンテーション＋体積推定AI |
| 振動板面形状（ドーム/コーン） | 3Dポイントクラウド + ディープラーニング |
| マグネット磁場均一性 | 磁場センサーアレイ + アノマリー検知 |
| 完成品音響スクリーニング | FFT＋ML分類器（不良ランブル検出） |

AI検査システムは2024〜2025年にかけてエージェント自動化市場として約3億ドル→9億ドルへ3倍の成長を記録した（出典：[DXtech.jp](https://dxtech.jp/ja/ai-automation-trends-2025-2026/)）。

### スピーカードライバー製造フロー（現代の自動化版）

```
① 磁気回路組立：フェライト/ネオジム磁石 → ポールピース＋プレートへの圧入（サーボプレス）
② ボイスコイル巻線：自動巻線機（μm精度、エナメル線0.05～0.3mm）
③ スパイダー＋ボイスコイル接着：UV硬化またはエポキシ接着（ロボット塗布）
④ 振動板取付け：前面コーン or ドームの精密位置合わせ（機械ビジョン）
⑤ リード線半田：レーザーはんだ（Goertek方式）または自動半田ロボット
⑥ 特性検査：周波数特性・Rdc・磁束密度・THD（自動試験機）
⑦ AIビジョン最終外観検査
⑧ 梱包（この工程のみ多くが人手）
```

## 📄 論文・特許・研究 {#論文}

### Bluetooth スピーカーのLCA（ライフサイクルアセスメント）研究

PMC（PubMed Central）掲載論文「製品ライフサイクルに基づく炭素フットプリント計算手法の開発：STAGEブルートゥーススピーカーの事例研究」（[PMC](https://pmc.ncbi.nlm.nih.gov/articles/PMC12367186/)）では、低炭素設計を目的としたスピーカー製品のライフサイクル炭素フットプリント計算手法が提案されている。STAGEブルートゥーススピーカーを事例として、製造・輸送・使用・廃棄の各フェーズにおける炭素排出量を定量化し、製造フェーズが炭素排出の主要因であることを明らかにしている。

### Node Audio MonoCell＆HTL特許

Node AudioはAtomに搭載された**MonoCell制振ラティス構造**と**Helical Transmission Line（HTL）**の両方について特許取得を完了しており、アディティブマニュファクチャリングを前提とした音響エンクロージャー設計の特許事例として注目される（[DEVELOP3D](https://develop3d.com/3d-printing/under-the-covers-of-node-audios-atom-speaker)、[3DPrint.com](https://3dprint.com/284011/3d-printed-speakers-an-overview/)）。

### Selective Laser Sintering（SLS）によるコンパクトスピーカー

SLS（選択的レーザー焼結）を利用したPA12ナイロン製エンクロージャーは、MDF加工では不可能な内部音道の一体成形を実現する。小規模生産・カスタム設計に向いた製法として研究・製品化が進んでいる。

## 🔮 未来予測 {#未来}

**2026〜2030年の製造技術シナリオ：**

1. **AIビジョン検査の全工程化**：現在は特定工程（外観、巻線）に限られているが、音響スクリーニングを含めた全数AI検査が標準になると予想。不良率は現在比さらに50〜80%低減が見込まれる。

2. **3Dプリントの中量産進出**：FDM・SLSから、より高スループットな**Multi Jet Fusion（MJF）**や**Continuous Liquid Interface Production（CLIP）**へ移行することで、年産数万台規模の3Dプリントスピーカーが実現可能になる。

3. **ベトナムの高付加価値化**：現時点での低付加価値アセンブリからBEMシミュレーション・DSP開発まで含めた「垂直統合型ODM」へ進化するポテンシャルがある。ハノイ科技大学とGoertekが締結した産学連携はその先行事例。

4. **中国内製造の高度化**：生産コストの上昇に伴い、中国拠点は単純アセンブリから「AI設計＋自動化特殊プロセス」を中核とするハイエンド特化にシフトが加速する。GoertekのApple向け精密部品製造がそのモデルケース。
