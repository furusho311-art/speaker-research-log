## 特集：MEMSスピーカー量産元年2026 {#特集}

### 背景：なぜ今MEMSスピーカーなのか

スピーカードライバーの基本原理は100年以上変わっていない。コイル・磁石・振動板による動電型（電動型）ドライバーは、スマートフォン、イヤホン、スマートウォッチに至るまで支配的な存在であり続けた。しかし2024〜2026年にかけて、ついでその構造を根底から覆す技術が量産フェーズに突入している。それが **MEMS（微小電気機械システム）スピーカー** だ。

シリコン基板上に半導体プロセスで製造されるMEMSスピーカーは、動電型ドライバーと比べて体積70〜90%削減・重量90%削減を実現しうる。さらに固体構造による高い耐久性（IP58防水・10,000G耐衝撃）、温度特性の安定性、量産時のチップレベル一貫製造という製造上のメリットも大きい。

2026年は複数の主要プレイヤーが量産体制を整えており、業界では「MEMSスピーカー量産元年」と呼ばれ始めている。

### 1. xMEMS Cypress ── 世界初フルレンジMEMSスピーカーが量産へ

**[xMEMS Cypress 量産体制確立発表](https://www.businesswire.com/news/home/20250909860435/en/xMEMS-Announces-Mass-Production-Readiness-of-Cypress-The-Worlds-First-Full-Range-MEMS-Speaker-for-Wireless-Earbuds)**（2025年9月）

xMEMS Labs（米カリフォルニア州）は2025年9月、ノイズキャンセリング対応TWS（完全ワイヤレス）イヤホン向け世界初のフルレンジMEMSスピーカー「Cypress」の量産体制確立を発表した。顧客への量産出荷は2026年前半を予定している。

**Cypressの主要スペック：**

| 項目 | 仕様 |
|------|------|
| パッケージ体積 | 46mm³ |
| 重量 | 98mg |
| 最大SPL | 140dB以上（20Hzまで） |
| 低音域 | 従来比40倍の低周波音圧 |
| 動作原理 | 超音波パルス変調（空気を可聴帯域で動かさない） |
| 構造 | シリコン振動板、モノリシックソリッドステート |

Cypressの最大の技術的革新は動作原理にある。従来の動電型ドライバーが可聴帯域で振動板を前後に動かして空気を動かすのに対し、CypressはPiezoMEMS技術によって超音波パルスを生成し、それをオーディオ信号に変調して音を再生する。これにより、従来型ドライバーが苦手としていた近ゼロ位相特性、超高速の機械応答速度、優れた時間軸解像度が実現する。

ANCシステムとの親和性も高い。ドライバー自体が可聴帯域の機械振動を持たないため、ANCアルゴリズムとの干渉が少なく、より広帯域・高精度なノイズキャンセリングが可能になる。

**[xMEMS × Rayking 戦略提携：Cypress+Alta-Sターンキーモジュール](https://xmems.com/press-release/xmems-and-rayking-partner-on-cypress-alta-s-module-to-accelerate-adoption-of-the-worlds-first-full-range-mems-speaker-in-wireless-earbuds/)**

xMEMSは中国・東莞のRayking Electronics社との提携を通じ、Cypress + Alta-S ASIC（専用ドライブIC）を一体化したターンキーモジュールを共同開発。これにより、TWS OEMメーカーはMEMSスピーカー採用の技術的ハードルを大幅に下げられる。量産出荷は2026年初頭に開始予定で、同モジュールはxMEMS Live Asiaに合わせてサンプル提供が始まっている。

### 2. xMEMS Sycamore ── AIウェアラブル向けMEMSラウドスピーカー

**[Sycamore ── 世界初1mm薄型フルレンジMEMSマイクロスピーカー](https://xmems.com/press-release/xmems-introduces-sycamore-the-worlds-first-1-mm-thin-near-field-full-range-mems-micro-speaker-for-smart-watches-xr-glasses-and-goggles-open-fit-earbuds-and-other-applications/)**

Sycamoreは厚さ1.28mm・重量150mgの超薄型MEMSラウドスピーカーで、従来のコイルスピーカーに比べ体積70%削減・重量90%削減を実現する。設計対象はTWSイヤホンとは異なり、XRグラス・スマートウォッチ・オープンフィットイヤホン・AIウェアラブル向けのニアフィールド用途だ。

CES 2026（ラスベガス、The Venetian Suite 34-208）ではSycamoreを搭載したAIグラスプロトタイプが公開された。フレーム内にSycamore（フルレンジ音声）とµCooling（オンデバイスAI処理時の熱管理）を統合し、AIワークロード中の表面温度を体感できるレベルに抑える設計となっている。

**[Sycamore-W ── スマートウォッチ・フィットネスバンド向け派生モデル](https://xmems.com/press-release/xmems-unveils-sycamore-w-the-worlds-thinnest-speaker-engineered-for-smart-watches-and-fitness-bands/)**（2025年5月発表）

| 項目 | 仕様 |
|------|------|
| 外形寸法 | 20×4×1.28mm |
| 重量 | 150mg（従来コイル型 最大3gから） |
| 防水・防塵 | IP58 |
| 耐衝撃 | 10,000G |
| 体積削減 | 従来比70% |
| 量産時期 | 2026年Q2（早期アクセス提供中） |

スマートウォッチではスピーカーがケース内部に占める体積が大きく、バッテリーや生体センサーの搭載スペースを圧迫してきた。Sycamore-Wはスピーカーパッケージ体積を70%削減することで、設計者がより大容量バッテリーや多様なバイオセンサーを搭載できる余裕をもたらす。

### 3. SonicEdge SonicTwin ── スピーカーとマイクロフォンを1チップに統合

**[SonicEdge SonicTwin ST100：世界初MEMS スピーカー＋マイクロフォン一体型ソリューション](https://sonicedge.io/press-release/worlds-first-mems-speaker-microphone/)**

イスラエルのスタートアップSonicEdge（旧社名からリブランド）はCES 2025でSonicTwin（ST100）を発表した。これはTWS・ヘッドホン向けに、スピーカーとマイクロフォンを8×4×1mmの単一パッケージに統合した世界初の製品だ。

**SonicTwinの革新ポイント：**

- **変調超音波技術**（特許取得済み）により、従来型の機械振動を廃除
- スピーカーとマイクの物理的分離が不要になり、TWS筐体内の設計自由度が大幅に向上
- 部品点数の削減で製造ライン複雑度を低減し、タイムトゥマーケットを短縮
- 高SPLとin-earカナル統合を両立

エンジニアリングサンプルは2025年中に提供開始、コンシューマー製品への搭載は2026年を予定している。

**[SonicEdge CES 2025 詳細レポート（audioXpress）](https://audioxpress.com/news/sonicedge-presents-integrated-mems-speaker-and-microphone-solution-at-ces-2025)**

## 技術トレンド {#技術トレンド}

### MEMSスピーカーの4大アーキテクチャ

MEMSスピーカーは動作原理によって以下の4種類に分類される。

**1. 静電型（Electrostatic）**
微小な平行板電極間の静電引力で振動板を駆動。高周波特性に優れるが低音域の音圧確保が課題。

**2. 圧電型（Piezoelectric）**
圧電材料（PZT、AlNなど）に電圧を印加して機械変形を得る。xMEMS CypressとSycamoreが採用するPiezoMEMS方式はこのカテゴリに属し、超音波変調により可聴帯域全域をカバーする。

**3. 電動型（Electrodynamic MEMS）**
磁石とコイルをMEMSスケールで実装。従来の動電型と同原理だがシリコン基板上に形成。

**4. 熱音響型（Thermoacoustic）**
電流によるジュール熱で近傍空気を膨張・収縮させ音波を生成。可動部ゼロの特殊方式。

現在量産に最も近いのは圧電型（PiezoMEMS）で、xMEMSが最先端を走っている。SonicEdgeの変調超音波技術も圧電型の変形と位置付けられる。

### µCooling ── スピーカーと一体化する熱管理

xMEMSが提唱する「µCooling（マイクロ冷却）」は、AIウェアラブルにおける熱問題を解決するファン・オン・ア・チップ技術だ。AIグラス等のオンデバイス推論ではSoC（システム・オン・チップ）が大きな熱を発生させるが、µCoolingはフレーム内の微細流路で熱を排出し、表面温度を40%低減・熱マージンを70%改善する。スピーカー（Sycamore）と冷却（µCooling）を同一MEMSファミリーで統合するxMEMSのアプローチは、AIウェアラブル設計において競合他社との差別化となっている。

## 未来予測 {#未来予測}

### 2026〜2028年：MEMSスピーカーが当たり前になる世界

現在のMEMSスピーカーはいくつかの課題もある。Cypressの変調超音波方式は高精度なデジタル回路とAsicが必要で、動電型より信号処理が複雑だ。また価格も量産初期フェーズでは動電型より高価になる見込みだ。

しかし半導体量産の論理は強力だ。TSMC等の先端ファウンドリでウェーハレベルに製造されるMEMSスピーカーは、量産台数が増えれば急激にコストが低下する。スマートフォン向けMEMSマイクロフォンが2000年代初頭から急拡大し現在は事実上全スマホに搭載されているのと同じ軌跡を、MEMSスピーカーが2026〜2030年にかけて辿ると予想する専門家は多い。

2026年はその出発点となる「MEMSスピーカー量産元年」として、後から振り返れば技術史のターニングポイントになるだろう。
